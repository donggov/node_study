# node_study
nodejs 스터디

1. 환경
 - express
 - ejs
 - mongoDB
 - nodemon(개발용) / pm2(운영용)

2. 환경 세팅
 - npm install -g nodemon
 - npm install -g pm2
 - npm install -g express-generator
 - npm install
 - npm install --save mongodb
 - npm install --save ejs-locals

3. mongDB 실행
 - mongod --dbpath c:\MongoDB\data (data path 설정 옵션)
